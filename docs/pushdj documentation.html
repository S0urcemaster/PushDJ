PushDJ Documentation

PushDJ is a software MIDI transceiver to connect Ableton Push to Native Instruments' Traktor. It aims on improving the workflow of mixing music compared to a common DJ controller. Originated more from necessity, it can or already has functions and capabilities not found on a common DJ controller and tries to push Traktor to its limit.

Setup

  1. Start Traktor and import PushDJ.tsi. Close Traktor again.

  2. Start PushDJ.jar. Push's display and buttons change.

  3. Start Traktor. Push's display changes again.

  4. Load a track into A and push the red button on the bottom left to play the track.

Troubleshooting

Connect Ableton Push to your PC and start PushDJ.jar, if your windows recognizes executable jar files or start.bat if not. PushdJ.jar has the cleaner start as it doesn't open a console/cmd window. The additional console windows will display errors. Use it if you encounter problems.

The graphical user interface is unused at the moment. Any function of the buttons is not intended and should not be used.

After the PushDJ window pops up, the Push device's buttons should change color and the display its text. The push buttons and knobs are not working as long as Traktor is not running. When starting Traktor, Push's display will change again according to the MIDI outs set in Traktor's controller manager.

Locate PushDJ.tsi in PushDJ's directory and import it in Traktor's controller manager:

Add → Import TSI → Import other

At the moment, Traktor has problems detecting the virtual MIDI device set up by PushDJ when Traktor is running. Restart Traktor to make it detect the PushDJ device. In controller manager there should be a device called

Generic MIDI (PushDJ)

If you select it, the IN and OUT port should display

PushDJ

There were situations where Push was unresponsible even the setup was correct, then it helped to turn Push off and on again.

Controlling Traktor

PushDJ is in alpha state and functions will be added in the future.

Overview


PushDJ is planned to controlling all 4 Traktor decks as Track, Remix and Stem decks. In it's current early state only the first 3 decks and only the track deck type is implemented. As to Push's nature, any function of Traktor that is best used with faders like volume is supposed to be used with a different controller. Myself, i'm using a Vestax VCM-600 controller which is usually used for production like Ableton.

Volume, EQs and filter knobs and faders are not implemented in PushDJ and are not planned to be. For mixing music it is stringently required to use another controller together with PushDJ.

For the effects section of Traktor, i consider the advantages of implementing it on Push but as it is now, the effects are easily mapped on my vestax controller and I regard it „ok“ for use in my mixes.

Functions that will most likely not change

The biggest advantage of PushDJ compared to a common DJ controller is the opened up MOVE and CUE section of Traktor. While there are often only 4 cue buttons per deck and the other 4 are only accessible through a shift button as well as the MOVE section is completely unpresent or only accessible through shift buttons as well, Ableton Push has a lot of button to map these functions to.

2- Decks view

The Push's button matrix is split in two in a left and right side which will display the functions for a Traktor deck. The only possible layout for a deck at the moment is a track deck. Each of the sections of a track deck on Push will be now explained:

Start/Stop, Cue, Sync, Master

Simply accords to the Traktor functions and common DJ controllers. The Sync button just taps the phase to zero. To permanently turn on/off Sync use PushDJ's Shift button which is the Play button on push.

8 Cue points

This is the common use of cue points. Pressing a button will jump to the existant cue point or create a new one. To delete a cue point, use PushDJ's Delete-Shift button, which is the Record button on Push.

4 Jog Turn buttons

You are not required to use a common DJ controller together with PushDJ. It is intended to have only another controller for the volume fade and EQ section. So the Jog Turn function will be missing if you don't want to mix in sync mode and for positioning off-beat. So, the 2 middle buttons are for fine jog and the outer buttons for coarse jog movement.

BeatJump and Cue Loops

You can set the BeatJump size, of the loop and its IN and OUT points directly wtih Push's „resolution“ buttons right beside the button matrix. The size applies to any movement and any deck. The ability in Traktor to set the move size for any deck and loop and BeatJump separately is being considered oversized and complicated.

There are 2 move buttons for left and right direction each for BeatJump, Loop in its whole, Loop In and Loop Out. The Loop In and Loop Out buttons are not present instead there are 2 buttons for setting and activation a loop:

Loop Size Select + Set
Backward Loop Size Select + Set
